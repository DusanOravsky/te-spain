Liquidaciones — month-end builder

Pick the three monthly files; the app pastes the new fleet, computes price + consumption averages, appends the new row to Liquidaciones de combustible, and gives you the email snippet. All processing happens in your browser.

Step 1

Vendor fleet file (Flota Viva)
Single sheet, headers on row 10, data from row 11 onward.

Step 2

Liquidaciones workbook
Will be updated with the new fleet + a new month row. Original file is not changed; you'll get a new file to download.

Step 3

Fuel prices file (preciosEESS_es.xls)
Download the latest from geoportalgasolineras.es, then pick it here. Average Precio gasolina 95 E5 and Precio gasóleo A are computed across all stations that report a price.

Step 4 — settings

Defaults to today. Format matches the existing rows: dd/MM/yyyy.
